\begin{resumo}
O presente trabalho descreve o desenvolvimento do \textit{Prism}, uma aplicação mobile voltada à recomendação personalizada de filmes, projetada para facilitar a descoberta de conteúdo em plataformas de streaming. A solução utiliza o framework \textbf{Flutter} para a construção da interface e adota o \textbf{Firebase} como infraestrutura de back-end, integrando serviços como \textit{Authentication} e \textit{Cloud Firestore}. Para ampliar a qualidade das sugestões, o sistema consome metadados da \textbf{TMDb API} e emprega a LLM \textbf{Gemini Flash 2.5}, acessada por meio do \textbf{OpenRouter}, responsável por interpretar preferências do usuário e gerar recomendações com técnicas de linguagem natural. As telas foram projetadas no \textbf{Figma}, garantindo uma navegação clara e consistente entre dispositivos. O resultado é uma aplicação funcional, segura e escalável, que demonstra a viabilidade da integração entre IA generativa e tecnologias \textit{serverless} na construção de sistemas modernos de recomendação.
\end{resumo}
